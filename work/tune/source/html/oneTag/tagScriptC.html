Service Key가 없다면?

  1. TUNE720( tune720.com ) 광고주 회원 가입
  2. 원태그 메뉴 이동
  3. 사용중인 도메인 입력 후 만들기를 클릭하면 스크립트 생성
  4. 우측 상단 ‘도메인 정보’ 클릭 후 해당 도메인의 Service key 복사
  5. 카페24에서 다시 원태그 설치하기 또는 원태그 앱에서 '관리하기' 클릭 후 입력
