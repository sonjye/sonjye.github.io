Service Key가 없다면?

1. TUNE720 광고주 회원 가입

2. 로그인 후 원태그 메뉴 이동

3. 사용중인 도메인 입력 후 만들기를 클릭하면 스크립트 생성

4. 우측 상단 ‘도메인 정보’ 클릭 후 해당 도메인의 Service key 복사

5. 메이크샵스토어 > TUNE720에서 Service Key 입력 후 ‘시작하기’ 클릭
